About Block Explorer

Greenshilling Block Explorer is an open source web tool that allows you to view information about blocks, addresses, and transactions on the Bitcoin blockchain. The source code is on GitHub.

If you are new to Green Coin, check out We Use Coins and Greenshilling.org.

Public Greenshilling Coin API: Machine readable stats & blockchain info can be accessed directly through the REST and Websockets APIs.

Testnet is Greenshilling Coin sandbox. Block Explorer supports viewing both the testnet and mainnet blockchains.

Thanks to Private Internet Access for hosting the site. They provide a VPN Service that accepts Bitcoin.
